
> Getting Started
> https://webpack.js.org/guides/getting-started/

# install webpack
```
$ npm init -y
$ npm install --save-dev webpack
```

# start(compile)
```
$ webpack
```

# compile command
```
$ npx webpack _assets/js/index.js --output dist/js/bundle.js
```

# after set config file.
```
$ npx webpack --config webpack.config.js
```

```
$ webpack
```
